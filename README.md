# 🍄 Mario Jump Game

Um mini jogo inspirado no clássico do Mario, desenvolvido com **HTML, CSS e JavaScript**, onde o objetivo é simples: **pular os obstáculos e sobreviver o máximo possível**.

Este projeto foi criado com base em um tutorial do YouTube, com foco em praticar lógica de programação, manipulação do DOM e eventos de teclado.

---

## 🚀 Funcionalidades

* 🕹️ Personagem controlado por tecla (pulo)
* 🚧 Obstáculos em movimento
* 💥 Detecção de colisão
* 🔁 Reinício automático ao perder
* ⚡ Animações com CSS

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (animações com `@keyframes`)
* JavaScript (DOM, eventos e lógica do jogo)

O jogo utiliza conceitos básicos de desenvolvimento web, como manipulação de elementos e detecção de colisões, comuns em jogos simples feitos com JavaScript ([Tutorials.Vyeron.com][1]).

---

## 🎯 Como jogar

1. Abra o arquivo `index.html` no navegador
2. Pressione qualquer tecla (ou barra de espaço) para fazer o Mario pular
3. Desvie dos obstáculos
4. Tente sobreviver o máximo de tempo possível
